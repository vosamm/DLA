  • 설정 변경: 각 모니터의 설정 버튼 → 타입 및 무시 라인 수 조정 후 저장

changedetection.io 고급 설정

localhost:5000에서 watch별로 더 세밀한 설정이 가능합니다.

  • 시각적 필터 선택기: 마우스로 감지할 영역을 직접 클릭해서 지정. 헤더·광고·사이드바를 통째로 제외할 수 있어 무시 라인 수보다 정확합니다.
    Watch 편집 → Visual Filter Selector (또는 CSS/XPath Selector 직접 입력)
  • 체크 주기 변경: Watch별로 1분~수 시간 단위로 조정 가능
    Watch 편집 → Time between checks
  • 텍스트 포함/제외 필터: 특정 키워드가 있을 때만 알림
    Watch 편집 → Filters & Triggers → Text to filter
  • JavaScript 실행: 페이지 로드 후 JS를 실행해서 로그인·클릭 등 처리
    Watch 편집 → Request → Execute JS
